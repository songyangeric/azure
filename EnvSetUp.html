How to set up Azure Python environment

  1. Install pip (You may need to use sudo )
curl "https://bootstrap.pypa.io/get-pip.py" -o "get-pip.py"
python get-pip.py

  1. Install cryptography (this is suggested although following procedures will do this implicitly)

    • For Debian and Ubuntu, the following command will ensure that the required dependencies are installed:
      sudo apt-get install build-essential libssl-dev libffi-dev python-dev
    • For Fedora and RHEL-derivatives, the following command will ensure that the required dependencies are installed:
      sudo yum install gcc libffi-devel python-devel openssl-devel

    You should now be able to build and install cryptography with the usual.(You may need to use sudo on Ubuntu)

      pip install cryptography

  2. Install azure Python SDK

    • Via pip(You may need to use sudo )
      Use the –pre flag: 

       pip install --pre azure

      Specify the version:

       pip install azure==2.0.0rc6

      You can also install only what you exactly need:

      Use the –pre flag: 

       pip install --pre azure-mgmt-compute

      Specify the version: 

      pip install azure-mgmt-compute==0.30.0rc6
    • Via source codes(You may need to use sudo)
      git clone git://github.com/Azure/azure-sdk-for-python.git
      cd azure-sdk-for-python
      python setup.py install
Issues you may encounter:

  1. On Ubuntu: Err http://security.ubuntu.com/ubuntu/ ** 404 Not Found
    Please update your repo first:

     sudo apt-get update

  2. ImportError: You need to install ‘msrest’ to use this feature
    Run command(You may need to use sudo):

   pip install msrest
  1. ImportError: You need to install ‘msrestazure’ to use this feature
    Run command(You may need to use sudo):
   pip install msrestazure
  1. ‘X509’ object has no attribute ‘_x509’
    Please upgrade your pyopenssl to a newer version.
    Run command(You may need to use sudo):
  pip install pyopenssl
